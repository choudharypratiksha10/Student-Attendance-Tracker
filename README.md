# Student-Attendance-Tracker
Python Project

Statement: "Develop a Python program to maintain attendance records for students in a class, including marking present, absent, and generating reports."

A web-based attendance management system built using **Flask, HTML/CSS, and Excel as a database**.  
This project allows teachers to manage students, mark attendance with date, filter records, and generate PDF reports.

## 🚀 Features
- ➕ Add Student (Name, Roll Number, Department)
- ✏️ Edit Student Details
- ❌ Delete Student (with confirmation popup)
- 📅 Mark Attendance with Date
- 🔍 Search by Name or Roll Number
- 🎯 Filter by Department
- 📊 Generate Attendance Report
- 📄 Download PDF Report (Filtered by Department & Date)
- 📱 Responsive UI (works on mobile & desktop)
- 📂 Excel used as Database

## 🛠️ Technologies Used
- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS  
- **Database:** Excel  
- **PDF Generation:** `reportlab`
